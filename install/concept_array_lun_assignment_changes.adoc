= アレイLUNの割り当て変更
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
_spare_array LUNの割り当ては、あるONTAPシステムから別のシステムに変更できます。所有権の変更は、ノード間で負荷を分散するために必要になる場合があります。
