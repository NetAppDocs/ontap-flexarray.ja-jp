= アレイLUNへのパスの計画
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
パスは、ONTAPシステムとストレージアレイ間の物理接続です。ONTAPシステムとストレージアレイの間にSingle Point Of Failure（SPOF；単一点障害）が発生しないようにするには、冗長パスが必要です。
